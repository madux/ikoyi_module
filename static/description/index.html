Create & Manage Material Orders.

Go to Inventory → Material Order Form


  • Create an MOF
  • Send to Manager
  • Manager Approves to Procurement

    • Click procurement >> Purchase Orders

    You will see the orders created. Open one order to start Processes as follows


  • Generate or Cancel Request
  • It will take you to the Request Module view.
    (For later purposes: You can go to the request module to view it)
  • The procurement manager approves → The internal control dept. verifies and approves
  • The General manager approves → procurement manager verifies and creates LPO
  • It will take you to the Procurement Module view.
    (For later purposes: You can go to the request for quotion module to view it)
  • On the Request for Quotation view, click confirm order button → It will generate a confirmed LPO
  • Click on the Receive products or go directly to the inventory to recieve and validate your orders.
  • Account Payable officer Goes to the request module and creates vendor bill upon notification

    • (Note: Account payable office can use the Menu under Accounting → Payment & Requisition → Account Requisition or Requisition form)

    Accounts prepares Requisition Form

    The next process is as follows:


  • Go to Accounting → Requisition Form or Account Requisition Menu
    (This can also be done from the Request → Requisition Form
  • Add the vendor name, and all records will be populated
  • Select the bill generated for the vendor
  • Send to accountant
    (You can also print and view vendor bills as well)
  • After all approvals , it creates a payment mandate
  • Click on the payment mandate and continue with approvals
  • Click on the view invoice, to register vendor bills payments thereafter
